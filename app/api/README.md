# App API Routes (Scaffolding)

This directory is **scaffolding for a potential future migration** from the Pages Router (`pages/api/`) to the Next.js App Router (`app/api/`).

## Status

**DO NOT USE.** All directories in this folder are empty placeholders. No actual endpoints are implemented here.

## Future Plan

The application may eventually migrate API routes from `pages/api/` to `app/api/` to align with modern Next.js patterns. Until that migration happens:

1. **All new endpoints should be created in `pages/api/`**
2. **Do not assume any routes in this directory are functional**
3. **See `pages/api/README.md` for the actual API surface**

## When Migration Happens

Once a migration to App Router begins, this README should be updated with:
- New endpoint locations
- Migration checklist
- Deprecation timeline for Pages Router

Until then, treat this directory as a placeholder for future work.
